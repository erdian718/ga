# ga

[![GoDoc](https://godoc.org/github.com/ofunc/ga?status.svg)](https://godoc.org/github.com/ofunc/ga)
[![License](http://img.shields.io/badge/license-MIT-red.svg)](https://github.com/ofunc/ga/blob/master/LICENSE)

Package ga implements the genetic algorithm. It can handle negative fitness properly. Mutation probability is adaptive and does not need to be set.
